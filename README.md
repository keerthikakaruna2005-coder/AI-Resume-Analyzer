# AI Resume Analyzer

## Project Description

AI Resume Analyzer is an NLP-based application that analyzes resumes and provides career insights.

The system extracts skills, calculates resume score, recommends job roles, and suggests improvements.

## Features

- Resume PDF upload
- Text extraction
- AI skill detection
- Resume scoring
- Missing skill identification
- Job role recommendation
- Skill improvement suggestions

## Technologies Used

- Python
- Streamlit
- NLP
- Machine Learning
- PyPDF2
- Scikit-learn
